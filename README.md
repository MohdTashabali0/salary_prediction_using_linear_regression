# salary_prediction_using_linear_regression
A simple machine learning project featuring the concept and application of linear regression.
This project implements a Machine Learning model to predict salaries based on years of experience using Linear Regression. The model is trained using a dataset and evaluated with multiple metrics to measure performance.
📌 Project Overview
Salary prediction is a common regression problem in Machine Learning.
In this project, we use Simple Linear Regression to learn the relationship between:
Years of Experience (Independent Variable)
Salary (Dependent Variable)
The model learns from historical salary data and predicts the salary for a given number of years of experience.
🚀 Features
Data preprocessing using Pandas
Train/Test dataset splitting
Model training using Scikit-Learn
Performance evaluation using:
R² Score
Mean Squared Error (MSE)
Root Mean Squared Error (RMSE)
Data visualization using Matplotlib
Salary prediction for new experience values
🧠 Machine Learning Algorithm Used
Linear Regression
Linear Regression predicts a continuous value by fitting a straight line between the independent and dependent variables.
Equation:
y = mx+c
Where:
y = Predicted Salary
x = Years of Experience
m = Slope of the line
c = Intercept

Python libraries used: 
Pandas
Numpy
Matplotlib
Scikit-learn

Dataset from kaggle: https://www.kaggle.com/datasets/karthickveerakumar/salary-data-simple-linear-regression
